Getting Started: Running the “PIC32MX USB Starter Kit - Generic Host” demo

 

 

Required Hardware

Firmware

Running the Demo

Required Hardware:

  To run this project, you will need 2 of the following sets of hardware, one for USB Host, one for USB Device:

 

   PIC32MX USB Starter Board (DM320003)

 

 

Firmware:

To run this project, you will need to load the corresponding firmware.

 

The source code for this USB host demo is available in the <Install Directory>\usb_sk_generic_host_demo directory. The source code for the USB device is available in the  <Install directory>\usb_sk_generic_device_demo directory.  In this directory you will find all of the user level source and header files as well as project and workspace files.  Find the project (*.mcp) or workspace (*.mcw) file.  Compile and program the demo code into the hardware platform.  For more help on how to compile and program projects, please refer to the MPLAB® help available through the help menu of MPLAB (Help->Topics…->MPLAB IDE).

 

1.       Build the USB device project and program the USB Device (PIC32MX Starter Board #1) for Release mode (refer to PIC32MX Starter Kit 1.1 Release Notes for details).

2.       Build the USB host project and program the USB Host (PIC32MX Starter Board #2) for debug mode.

 

 

Running the Demo:

This demo will require two Microchip devices to run.  One will act as the USB host and the other will run the USB peripheral. 

 

How to use:

1. Attach a USB cable to the debugger connector J1 on the USB Device (PIC32MX Starter Board #1) and the other end to an available USB port on the PC.  This provides power to the USB Device demo.

2. Connect the USB device demo (PIC32MX Starter Kit #1) to the Type-A USB receptacle on the USB Host (PIC32MX Starter Board #2).  LED1 will blink.

3. Press SW1 to display temperature and Potentiometer (POT) readings.

4. Press SW2 to light LED3 on the PICDEM(TM) FS USB Demo board (D9 on the Explorer 16 board).

5. Press SW3 to light LED4 on the PICDEM(TM) FS USB Demo Board (D10 on the Explorer 16 board).

(Output will appear the MPLAB PIC32 Starter Kit console window.)
