Getting Started: Running the “PIC32MX USB Starter Kit - Mass Storage Device” demo

Firmware

Running the Demo

 

Required Hardware:

  To run this project, you will need one of the following sets of hardware:

 

   PIC32MX USB Starter Board (DM320003)

   PIC32MX IO Expander Board (DM320002)

   SD Card PICTail Plus (AC164122)

 

Firmware:

To run this project, you will need to load the corresponding firmware into the device.

 

The source code for this demo is available in the “<Install Directory\sample_code\\usb_sk_msd_device_demo” directory.  In this directory you will find all of the user level source and header files as well as project and workspace files.  Find the project (*.mcp) or workspace (*.mcw) file.  Compile and program the demo code into the hardware platform.  For more help on how to compile and program projects, please refer to the MPLAB® IDE help available through the help menu of MPLAB (Help->Topics…->MPLAB IDE).

 

 

Running the Demo:

This demo uses the hardware platform as an SD card reader.  Connect the hardware platform to a computer through a USB cable. 

How to use:

Requires the use of a PIC32 IO Expansion Board and a PICtail(TM) Board for

SD(TM) and MMC Cards (as well as a FAT16/32 formatted SD card for storage).

1. Connect PICtail(TM) card as directed, in first position on PICtail(TM)

   expander board (so that pin-1 on Expansion Board receptacle contacts

   pin 1 on the PICtail(TM) board.

2. Attach device connection to (Micro-B connector) to host.  Device will

   enumerate as a Mass Storage Device.

 

The device should appear as a new drive on the computer named “Removable Drive”. 

 

If no SD Card is inserted in the SD Card PICTail Plus, the following dialog will pop-up.

 

Once a compatible card is inserted in the card reader, files can be read, deleted, and manipulated like any other drive on the computer.
