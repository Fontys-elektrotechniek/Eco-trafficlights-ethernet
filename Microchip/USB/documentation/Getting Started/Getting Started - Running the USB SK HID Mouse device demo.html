Required Hardware:

  To run this project, you will need one of the following sets of hardware:

 

PIC32MX USB Starter Board (DM320003)

 

 

Firmware:

To run this project, you will need to load the corresponding firmware into the devices.  There are two methods available for loading the demos: Precompiled demos and source code projects.

 

Precompiled Demos are available in the “<Install Directory>\USB Precompiled Demos” folders.  Each demo should be prefaced with the hardware platform it is compiled for.  Select the file that matches the hardware that you have and the demo that you want to run.  For more information about how to load a precompiled project, please see the “Getting Started – Loading a precompiled demo” guide.

 

The source code for this demo is available in the <Install Directory>\sample_code\usb_sk_hid_mouse_device_demo directory.  In this directory you will find all of the user level source and header files as well as project and workspace files.  Find the project (*.mcp) or workspace (*.mcw) file.  Compile and program the demo code into the hardware platform.  For more help on how to compile and program projects, please refer to the MPLAB® IDE help available through the help menu of MPLAB IDE (Help->Topics…->MPLAB IDE).

 

 

Running the Demo:

This demo uses the hardware platform as a USB mouse, allow you to move the cursor by pressing switches on the PIC32MX Starter Kit.  The demo also displays the status of the demo using the LEDs on the board.  The switches and LEDs are used as follows:

 

How to use:

SW1  - Move mouse cursor up   (momentary contact)

SW2  - Move mouse cursor Left (momentary contact)

SW3  - Invert direction of mouse move for SW1 & SW2 (momentary contact)

 

LED1 - Mouse Connection status (emulation is always enabled)

        LED On       - Mouse emulation enabled

        LED Off      - Only when board is not powered/running

        LED Blinking - Attached to USB

LED2 - Remote Wakeup Status

        LED On  - Remote wakup enabled

        LED Off - Remote wakup disabled

LED3 - Device sleep status

        LED On  - Device is asleep

        LED Off - Device is awake
